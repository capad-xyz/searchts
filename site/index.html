searchts.

The missing layer between AI and the web. Read bot-walled pages as clean Markdown, search, transcribe, and grab assets — keyless, free, on your own IP.

GitHub PyPI MCP Registry MIT License

Install

# CLI + library
pip install searchts

# or run without installing
uvx searchts read https://example.com

Four verbs

searchts read https://example.com          # any page as clean Markdown, through most bot-walls
searchts search "open source vector db"    # multi-provider web search, no API key
searchts transcribe https://youtu.be/...   # YouTube/TikTok/Instagram/Reddit transcripts
searchts grab https://example.com          # images/icons/css/fonts + color palette

read — escalating unlocker

Browser-fingerprint fetch → Jina Reader → stealth browser. Escalates only as far as needed and returns Markdown with a source receipt.

search — keyless search

DuckDuckGo by default with reciprocal-rank fusion; optional SearXNG, Exa, Brave, and Tavily providers.

transcribe — subtitles first

Existing captions via yt-dlp need no key at all; falls back to hosted or local Whisper.

grab — assets & design

Download a page's assets through the same unlock ladder and extract its color palette and fonts.

Give it to your agent (MCP)

# Claude Code
claude mcp add searchts -- uvx --from "searchts[mcp]" searchts mcp serve
